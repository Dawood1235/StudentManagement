# Student Management System (Python)

## 📌 Project Overview

The **Student Management System** is a simple Python-based project that allows users to manage student records using file handling. The system can store, update, and retrieve student information efficiently using a text file.

This project demonstrates the use of **Python programming fundamentals**, including file handling, loops, functions, and conditional statements.

---

## 🚀 Features

* Add new student records
* View existing student records
* Update student information
* Delete student records
* Store data in a text file for persistence
* Simple console-based interface

---

## 🛠 Technologies Used

* **Python**
* **Text File Handling (.txt)**
* Basic programming concepts (loops, conditions, functions)

---

## 📂 Project Structure

```
student-management-system/
│
├── main.py
├── stud_records.txt
└── README.md
```

* **main.py** – Main Python program containing the student management logic
* **stud_records.txt** – File used to store student records
* **README.md** – Project documentation

---

## ▶️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/YOUR-USERNAME/student-management-system.git
```

2. Navigate to the project folder:

```
cd student-management-system
```

3. Run the Python file:

```
python main.py
```
